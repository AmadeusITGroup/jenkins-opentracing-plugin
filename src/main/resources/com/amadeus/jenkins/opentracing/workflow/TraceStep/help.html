Add tracing information to the pipeline execution. Can be used with or without a body.
When called with a body it:
  • Executes the body under a new tracing span
  • Adds the passed parameters to this tracing span
  • Returns the value of the execution of the body
When called without a body it:
  • Does not create a new tracing span
  • Adds the information from the passed parameters to the currently active tracing span
  • Returns an object that can be used to customize the span further
